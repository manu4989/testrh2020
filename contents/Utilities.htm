Fuel Consumption Calculator

Estimate the Fuel Consumption of a ship for a route under various meteorological conditions. This calculation is based on the ship’s target curves set up in the system

(if there is no Target Curve set up for the selected ship, the system notifies you to create one before proceeding). This helps in route planning and also for financial purposes.

Select a ship, a period of time, and any other filters you want to apply.

Fill in the Operative Conditions of the ship which come from the:

estimated sea state [D]

distance

speed

fuel ROB [t] (received from automation systems on board or from manual input)

quantity of each Cargo Type on board (received from automation systems on board or from manual input)

When you fill in the quantity of each Cargo type, the system automatically calculates the total Tons of cargo [t] on board.

Click to move on to the next step.

Change Displacement [t] and Mid draft [m] if they are different than the system calculated.

Click to see the Fuel Consumption results.

Scheduling a Data Acquisition

Acquire new data from the onboard installation to update the archive.

Click at the bottom of the main MAN Cylinder Pressure page.

A new window opens.

Fill in the number of cylinder revolutions you want the system to read, and the date you want to import the data.

NOTE: The data acquisition date/time must be at least two hours in the future. This gives the HQ and on-board installations enough time to synchronize data.

Once scheduled, you can see the pending acquisition in the Scheduled acquisitions area of the page. It is moved to the Acquisitions list after it’s received from the ship.
